# Experiment 2.3.2 – Redux Shopping Cart

## Aim
Implement Redux for state management in a shopping cart.

## Setup & Run
```bash
npm install
npm start   # Runs on http://localhost:3000
```

## Features
- Redux Toolkit configureStore + createSlice
- Add / Remove / Update quantity operations
- Cart total computed with selectors
- Cart state persisted to localStorage
- Material UI components
- Redux DevTools compatible (install browser extension)
